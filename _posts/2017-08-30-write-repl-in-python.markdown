Amjith Ramanujam gave a great talk at [PyCon2017](https://www.youtube.com/watch?v=hJhZhLg3obk)
on creating REPL (Read|Evaluate|Print|Loop) applications in Python using the [prompt_toolkit](https://python-prompt-toolkit.readthedocs.io/en/master/) package.

A minimum working example with command history, auto-suggestion, auto-completion and code highlighting
took 15 lines of code (including 5 lines of imports). Check this out!
